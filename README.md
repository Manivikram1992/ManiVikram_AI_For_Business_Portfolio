## Reflection<br>

**What I learned across all modules**<br>
This course walked through the full arc of a machine learning workflow, starting with the fundamentals of supervised learning and ending with neural networks and NLP.<br>
Working through classification, SVMs, decision trees, dimensionality reduction, and deep learning back to back gave me a clear picture of when to reach for a simple model versus something more complex, and why that choice matters for real projects, not just accuracy scores.<br>

**Skills I developed**<br>
- Data preprocessing and cleaning, including handling missing values, feature scaling, and encoding categorical data<br>
- Model tuning through cross-validation, grid search, and hyperparameter selection<br>
- Evaluating models beyond accuracy, using precision, recall, ROC curves, and confusion matrices<br>
- Visualization of data distributions, decision boundaries, and model performance<br>
- Dimensionality reduction techniques like PCA to simplify high-dimensional data<br>
- Building and training basic neural network architectures<br>
- Applying NLP techniques for text classification and language tasks<br>

**How these projects prepare me for real-world ML tasks**<br>
Each project mirrored a step you'd actually take on the job, from cleaning messy data to picking the right model for the problem and tuning it properly instead of just accepting default settings.<br>
Having hands-on reps with SVMs, decision trees, ANNs, and NLP means I can match a technique to a business problem instead of forcing one model to fit everything.<br>
This directly supports the kind of work I do now in cloud and data engineering, where building reliable, well-tested pipelines around a model matters as much as the model itself.<br>
